# SootheNote

SootheNote is a functional application designed to support cancer patients through a calm, focused, and private experience.

This repository does not expose the full implementation.

## Overview

SootheNote is built with patients in mind.  
The goal is to reduce friction, stress, and cognitive load during treatment by providing a simple, supportive digital tool.

The app is actively developed and used as a real product, not a demo.
